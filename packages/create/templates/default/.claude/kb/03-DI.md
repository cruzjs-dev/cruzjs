# Dependency Injection (Inversify)

## Core Pattern

### 1. Injectable Service

```typescript
import { Injectable, Inject } from '@cruzjs/core/di';
import { DRIZZLE, type DrizzleDatabase } from '@cruzjs/core/shared/database/drizzle.service';

@Injectable()
export class NotesService {
  constructor(
    @Inject(DRIZZLE) private readonly db: DrizzleDatabase,
  ) {}

  async list(userId: string) {
    return this.db.select().from(notes).where(eq(notes.userId, userId));
  }
}
```

### 2. Module

```typescript
import { Module } from '@cruzjs/core/di';

@Module({
  providers: [NotesService],
  trpcRouters: {
    notes: notesRouter,  // Available as trpc.notes.*
  },
  events: [
    { event: NoteCreatedEvent, listener: sendNotificationListener },
  ],
})
export class NotesModule {}
```

### 3. Register in createCruzApp

```typescript
// server.cloudflare.ts
import { createCruzApp } from '@cruzjs/core';
import { CloudflareAdapter } from '@cruzjs/adapter-cloudflare';
import { StartModule } from '@cruzjs/start';
import * as schema from './database/schema';
import { NotesModule } from './features/notes';

export default createCruzApp({
  schema,
  modules: [StartModule, NotesModule],
  adapter: new CloudflareAdapter(),
  pages: () => import('virtual:react-router/server-build'),
});
```


## @Module Options

```typescript
@Module({
  providers: [
    MyService,                                          // Simple class (singleton)
    { provide: MyService, scope: 'transient' },         // New instance every resolve
    { provide: TOKEN, useClass: MyImpl },               // Interface binding
    { provide: TOKEN, useValue: someInstance },          // Constant value
    { provide: TOKEN, useFactory: (dep) => new X(dep), inject: [Dep] },  // Factory
    { provide: JOB_HANDLER, useClass: MyJob, multi: true },              // Multi-injection
  ],
  trpcRouters: {
    myFeature: myFeatureRouter,
  },
  events: [
    { event: MyEvent, listener: myListener },
  ],
})
export class MyModule {}
```

## Resolving Services in Routers

```typescript
import { getAppContainer } from '@cruzjs/core';

export const notesRouter = router({
  list: protectedProcedure.query(async ({ ctx }) => {
    const container = await getAppContainer();
    const service = container.resolve(NotesService);
    return service.list(ctx.session.user.id);
  }),
});
```

## Common Tokens & Services

| Token/Service | Import | Purpose |
|---------------|--------|---------|
| `DRIZZLE` | `@cruzjs/core/shared/database/drizzle.service` | Database instance |
| `ConfigService` | `@cruzjs/core/shared/config/config.service` | Env var access |
| `EventEmitterService` | `@cruzjs/core/shared/events/event-emitter.service.server` | Event dispatch |
| `StorageService` | `@cruzjs/core/shared/storage/storage.service` | File storage (R2/local) |
| `JOB_HANDLER` | `@cruzjs/core/jobs` | Job handler multi-inject token |
| `USER_HYDRATOR` | `@cruzjs/core` | User session data hydration |
| `OrgService` | `@cruzjs/start` | Organization management |
| `MemberService` | `@cruzjs/start` | Member management |

## Injection Examples

```typescript
// Database
@Inject(DRIZZLE) private readonly db: DrizzleDatabase

// Config
@Inject(ConfigService) private readonly config: ConfigService
this.config.getOrThrow<string>('APP_URL');

// Events
@Inject(EventEmitterService) private readonly events: EventEmitterService
await this.events.dispatch(new ItemCreatedEvent(item.id));

// Optional injection
@Inject(SomeService) @Optional() private readonly opt?: SomeService

// Multi-injection
@MultiInject(JOB_HANDLER) @Optional() handlers: JobHandler[] = []
```

## Scope Types

- **singleton** (default) -- one instance for entire app
- **transient** -- new instance every resolution

## CruzContainer Methods

```typescript
const container = await getAppContainer();
container.resolve(MyService);           // Get a service
container.resolveAll(JOB_HANDLER);      // Get all multi-injected
container.isBound(MyService);           // Check if registered
```

## Rules

1. Always use `@Injectable()` on service classes
2. Always use `@Inject(TOKEN)` for constructor params
3. Never instantiate services with `new` -- use DI
4. Register routers in `@Module`, not manually
5. Singleton scope for stateless services (default)
